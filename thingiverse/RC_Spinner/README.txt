# Summary

This is an unusual RC spinner that you can not find on the web, maybe I'm the first to create a similar object...why? Because you can use it with the propsaver (this one  https://www.hobbyking.com/en_us/3-17mm-prop-saver-set-blue-1pc.html)

I made this spinner for my homemade R/C combat aircrafts but you can print yours and install in your model if it goes well.

Info:
Diameter: 53 mm
Length: 44 mm
Hole: 7mm
Weigth: 16g
Screw:  2x  3mmx9.5mm

I tested the spinner with:
- 1000kv motor (2830 class)
- esc 20A
- 3s 1300mAh battery
- 8x6 propeller

In the video you can see the result.

# Print Settings

Printer: Anet A8
Rafts: No
Supports: Yes
Resolution: 0.1

Notes: 
**Infill**
- upper part: 100.  With supports
- base plate: 3 bottom solid layers, 50% honeycomb and 22 top solid layers. W/o supports

Perimeter shells: 2
Material: PLA 


# How to install

Here is a series of images to show you how to mount the spinner with your electric motor:

![Alt text](https://cdn.thingiverse.com/assets/2a/a2/f3/67/41/20161101_165004.jpg)
List of materials: spinner, propeller, 3x9.5mm screws, prop saver, rubber bands from bike inner tube, electric motor

![Alt text](https://cdn.thingiverse.com/assets/48/16/59/1f/07/20161101_164910.jpg)
Fit the propeller (w/o adapter) into the base plate

![Alt text](https://cdn.thingiverse.com/assets/12/4d/81/86/0b/20161101_164927.jpg)
Insert the prop saver into the rear part of the base plate

![Alt text](https://cdn.thingiverse.com/assets/fe/01/cc/bf/c3/20161101_164747.jpg)
Attach the rubber bands to the prop saver (it's a little bit hard due to the small holes in the plate)

![Alt text](https://cdn.thingiverse.com/assets/30/7d/ac/d5/b4/20161101_164623.jpg)

![Alt text](https://cdn.thingiverse.com/assets/35/c8/bb/8a/51/20161101_164551.jpg)
tighten up the screws

![Alt text](https://cdn.thingiverse.com/assets/6d/7e/87/62/70/20161101_164527.jpg)
tighten up the screws of the prop saver

![Alt text](https://cdn.thingiverse.com/assets/15/51/63/a6/b9/20161101_164342.jpg)
